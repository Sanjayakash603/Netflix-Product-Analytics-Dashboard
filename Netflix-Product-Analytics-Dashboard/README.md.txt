# Netflix Product Analytics Dashboard

## Project Overview
This Power BI dashboard analyzes Netflix content performance and user engagement metrics.

## Key KPIs
- Total Titles
- Movies
- TV Shows
- Countries
- Subscribers
- Retention Rate

## Dashboard Features
- Content Added Trend (Year-wise)
- Content Type Distribution
- Top 10 Genres
- Top 10 Countries by Content
- Rating Distribution
- Monthly Watch Hours Trend
- User Retention Rate
- Global Content Distribution

## Tools Used
- Power BI
- DAX
- Power Query
- Excel
- Data Modeling
